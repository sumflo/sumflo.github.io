About me...

Hi! I am Flora Konkoly-Sumegi. I am a - career changer - junior frontend developer. Although junior is a broad term nowadays, I would rather call myself a padavan.

After completing a seven-month Java & SpiringBoot backend developer bootcamp, I went from kindergarten teacher to frontend developer. Yeees... The force is with me.

I am proud to say that I have been working in my first development job for almost two years now, where I have learned from two amazing Masters and I hope to continue learning from them for a long time to come, and soon I'll be a true Jedi myself.

Core values & Principles

Clean code

I strive to write clean code that is easy to understand and maintain. Code quality is a top priority, as it gives efficient development and teamwork in the long term.

Reusability

...or laziness. The reusability of code is important. I separate repetitive elements and functions so that they can easily be used again.

Don't write a bug

Sometimes it's not easy, is it? But let's face it, there's no other way to write code that works. This lesson is crucial for quality work.

Keep learning

As a Padavan, it is essential that I constantly strive to learn new things and develop professionally. Learning never stops to keep up with technological changes.

Never give up

Commitment and perseverance are essential for me. I am determined and never give up, believing in the long-term results of hard work.

Always smile

A good atmosphere and a positive attitude are key to good performance. I always face challenges with a smile and I hope to share this energy with everyone.

About tihs project

This website was created using HTML and CSS. It's not really my thing, ( I mainly work with React and TypeScript) so you could say it was a challenge for me. I didn't use Bootstrap either, which is also not easy once you get used to it.

I started an online course 2-3 weeks ago and tried to use the knowledge I gained there to build this portfolio project. So... I tried to use grids instead of flexbox, because I've been using flex since the beginning, but it's hard to get out of the habit. Bottom line: it was a good exercise.

Pure HTML and CSS, no library, no framework, no scripting language... Back to basics with a focus on design - hope I didn't overdo it and everyone who sees it likes it. I tried not to include a bug :D

I know, I know... 'Do, or do not. There is no try.'
